This python script will extract data from <a href="fakepersongenerator.com">www.fakepersongenerator.com </a> and <b>add each new data to a csv file.</b><p>
The script requires <b>BeautifulSoup</b> libary to extract data. To install the library use <br/>
<b>pip install beautifulsoup4 </b> <p/>
Run the script using Python, the script will ask for number of rows to be added to the spreadsheet.
